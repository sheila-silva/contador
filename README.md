# Projeto Contator:
Este desafio de código foi proposto como uma das etapas de ensinamentos básicos da sintaxe ![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

Esse código Java define um programa de contagem numérica que lê dois números inteiros digitados pelo usuário, verifica se o primeiro é menor que o segundo.Caso seja, ele conta e exibe todos os números nesse intervalo (do menor ao maior). 

Esse programa é um exercício de controle de fluxo e tratamento de exceções em Java, útil para praticar:

* Entrada de dados com Scanner;

* Uso de try, catch e finally;

* Criação e lançamento de exceções personalizadas (throw e extends Exception);

* Laços for.

Neste desafio foram explorados conhecimentos em: 
* Declaração de classe e atributos.
* Métodos.
* Controle de fluxo com for e if.
* Manipulação de entrada de dados com Scanner.
* Operadores relacionais.
* Uso de System.out.print e System.out.println para saída de dados.
* Tratamento de exceções com a classe criada ParametrosInvalidosException 

*Este código foi criado e versionado respectivamente com:*
* IDE Intellij
* GIT

# Project: Counter

This coding challenge was proposed as part of the basic syntax lessons of  
![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

This Java code defines a **numeric counting program** that reads two integers entered by the user and checks whether the first number is smaller than the second one.  
If it is, the program counts and displays all the numbers within that range (from the smaller to the larger).

This program is an **exercise in flow control and exception handling in Java**, useful for practicing:

* Data input using **Scanner**  
* Use of **try**, **catch**, and **finally**  
* Creation and throwing of **custom exceptions** (`throw` and `extends Exception`)  
* **For loops**

In this challenge, the following concepts were explored:

* Class and attribute declarations  
* Methods  
* Flow control with **for** and **if** statements  
* Data input handling with **Scanner**  
* Relational operators  
* Use of **System.out.print** and **System.out.println** for output  
* Exception handling using the custom class **ParametrosInvalidosException**

* **IntelliJ IDE**  
* **Git**
